Goal of the final app

  • The app is for use by hill walkers who want to track and log their summits.
  • The app will include a database of summits with relevant info included for each.
  • Data included will be:
    • Name
    • Grid reference
    • Height above sea level
    • An image of the peak
    • Region
    • Hill type
    • Summited tickbox
  • When a user reaches the grid reference of a summit in the list, a notification will display
  • The database can be updated by a user to mark a peak as summited
  • The user can filter the database to identify peaks of interest to them based on their chosen criteria
  • The app includes advice for a mountain emergency situation
  • The app uses geo location data to display current grid reference to provide to emergency services
